# 🎮 Jogo em Python (Beta)

Um jogo de suspense feito em Python com sons para aumentar a tensão.

#  Sobre:
Esse jogo ainda está em desenvolvimento (beta), mas já possui sistema de perguntas e efeitos sonoros.

## ▶️ Como jogar
1. Baixe o projeto
2. Instale as bibliotecas:
   pip install pygame playsound
3. Execute:
   python jogo.py

#  recursos:
- Sons ambiente
- Sistema de perguntas
- Eventos aleatórios (em desenvolvimento)
